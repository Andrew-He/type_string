type_string is a string library built with C++ template,
and provide string "functions" for compiling time.

"functions" list ---
  struct u_strcat
  struct u_strlen
  struct u_substr
  struct u_get_char_at
  struct u_strchr
  struct u_strrchr
  struct u_strstr
  struct u_strrstr
  struct u_str_equal
